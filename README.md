# Forge_Text

Forge_Text is a simple and powerful React-based text utility application that helps users manipulate and analyze text quickly and efficiently.

## Features

- Convert text to **UPPERCASE**
- Convert text to **lowercase**
- Clear text instantly
- Copy text to clipboard
- Paste text from clipboard
- Remove extra spaces
- Live text preview
- Word and character counter
- Estimated reading time

---

## Tech Stack

- React.js
- Bootstrap (for styling)
- JavaScript (ES6)

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/mayank-garg7/forge_text.git
```

### 2. Navigate to the project directory

```bash
cd forge_text
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```

The app will run at:

```bash
http://localhost:3000
```

---

## Usage

1. Type or paste text into the textarea.
2. Use the buttons to manipulate the text.
3. View:

   * Word count
   * Character count
   * Estimated reading time
   * Live preview
